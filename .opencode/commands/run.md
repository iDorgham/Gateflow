Run the development server for the specified app. Ask the user which app to run if not specified.

Available apps:

- client-dashboard (port 3001)
- admin-dashboard (port 3002)
- marketing (port 3000)
- scanner-app (port 8081)

Use: pnpm turbo dev --filter=app-name
