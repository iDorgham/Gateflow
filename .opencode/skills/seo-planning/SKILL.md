# SEO Planning

## Content Calendar

- Map keywords to content (1 keyword cluster per page)
- Prioritize by intent + volume
- Include: title, target keyword, outline, CTA

## On-Page Structure

```markdown
# H1 — Primary keyword (one per page)
## H2 — Supporting topics
### H3 — Details
```

- One H1 per page
- H2/H3 in logical order
- Include target keyword naturally in H1 and first paragraph

## Internal Linking

- Link related pages (pillar → cluster)
- Descriptive anchor text (avoid "click here")
- Important pages linked from homepage/nav

## Output

- Content plan (topic, keyword, outline)
- URL structure / sitemap
- Internal linking map
