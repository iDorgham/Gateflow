# Content Creation

## Principles

- **Clear** — Short sentences, active voice, no jargon
- **Scannable** — Headings, bullets, short paragraphs
- **Goal-oriented** — Every section has a purpose

## Structure

```markdown
# Clear Headline

## Section (H2)
Brief intro. One idea per paragraph.

- Bullet when listing
- Keep items parallel

### Sub-section (H3) if needed
```

## Tone

- **Docs:** Neutral, instructional
- **Marketing:** Benefit-focused, confident
- **MENA:** Respectful, professional, consider RTL/AR

## Output

- Use markdown
- Include i18n keys for user-facing content (`t('key', 'fallback')`)
