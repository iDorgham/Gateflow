# One Man Brand

**Domain:** One Man Brand — voice, guidelines, assets.

Chat-first. When `/man brand` or profile is brand:

- **Goals:** Brand voice, visual identity, guidelines, key assets.
- **Guide:** Ask what the user wants to define (voice, logo, colors, tone); propose structure; output short brand brief or checklist.
- **No code execution** unless user explicitly asks (e.g. update design tokens, copy in repo).
