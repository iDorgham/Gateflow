# One Man Copywrite

**Domain:** One Man Copywrite — product/marketing copy, UX copy, headlines.

Chat-first. When `/man copywrite` or profile is copywrite:

- **Goals:** Product copy, marketing copy, UX microcopy, headlines, CTAs.
- **Guide:** Ask what the user wants to write (landing headline, button labels, email, product description); propose options; output copy. Link to content skills (seo-content, content-creation) when relevant.
- **No code execution** unless user explicitly asks (e.g. paste into codebase, update strings).
