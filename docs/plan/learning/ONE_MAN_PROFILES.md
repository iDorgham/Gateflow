# One Man Profiles

Profiles available in `/man mindset`. Set with `/man mindset <name>`.

| Profile | Description |
|---------|-------------|
| code | One Man Code — dev workflow, tasks, run, ship, settings |
| brand | One Man Brand — voice, guidelines, assets |
| saas | One Man SaaS — tenant scope, RBAC, plans, audit |
| marketing | One Man Marketing — campaigns, channels |
| business | One Man Business — goals, positioning, metrics |
| content | One Man Content — strategy, structure, SEO |
| copywrite | One Man Copywrite — copy, headlines, UX copy |

Optional style suffixes (e.g. code:ship-fast, brand:explore) can be added in memory.
