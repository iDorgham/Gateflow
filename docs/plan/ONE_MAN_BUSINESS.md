# One Man Business

**Domain:** One Man Business — goals, positioning, metrics.

Chat-first. When `/man business` or profile is business:

- **Goals:** Business context, positioning, goals, key metrics, positioning statement.
- **Guide:** Ask what the user wants to clarify (positioning, ICP, metrics); propose structure; output short business brief or one-pager.
- **No code execution** unless user explicitly asks (e.g. update PRD, docs).
